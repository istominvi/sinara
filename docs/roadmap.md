# Roadmap

## MVP
- Auth + роли: гость, учитель, ученик, админ платформы; базовый вход/регистрация и кабинеты.
- Методики + paywall: доступ к методикам для учителя, ограничение уроков без подписки, защита материалов.
- ДЗ + трекинг: выдача ДЗ из шаблонов методики, события просмотра/прослушивания/слайдов, прогресс.
- Словарь: слова урока и прогресс ученика.
- Расписание: занятия с учителем/группой, статусы и связка с уроками.
- Видеозвонок: встроенная комната занятия (Jitsi MVP).
- Сообщения: базовый чат учитель ↔ ученик/группа.
- Оплата: подписки учителя/школы (workspaces), учёт seats.
- Админка: минимальные настройки/управление доступом и оплатой.

## Phase 2
- Branding-контекст: workspace slug, темы/логотипы, контекст бренда для маршрутов `/s/<slug>`.
- Расширение админки: контроль контента, пользователей и аналитики.
- Углубление проверки ДЗ: принятие текстовых ответов учителем, более гибкие правила.
- Улучшение видеозвонка: LiveKit или расширенные настройки комнат.
- Дополнительные возможности: расширение словаря, отчётов и коммуникаций.
